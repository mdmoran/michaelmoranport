Reflections

Project Challenges

Acquiring a usable zombie image was difficult and none of the photos would load in correctly for me. With the help of a friend we were able to solve the issue using a media editing software. The leveling system was also difficult upon attempt, as I had never done it before and it was brand new to me.

Lessons Learned

How to make a basic game using JavaScript and how to debug the code associated with it.

Tools and/or Software Used

This was all created using:

  • JavaScript
  • p5.js Web Editor
  • GIMP

Next Steps

I plan to revisit this project to make it better. This would include a responsive screen size and a "play again" button, which I also had trouble with.
